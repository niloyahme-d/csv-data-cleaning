# Data Cleaning & Deduplication Automation (Python Demo)

**Problem it solves:** Client spreadsheets are often messy — inconsistent
capitalization, different phone number formats, duplicate entries, and
missing/invalid emails. Manually cleaning a few thousand rows by hand can
take days. This script does it in seconds.

## What it does
- Trims stray whitespace
- Standardizes text casing (Title Case)
- Normalizes every phone number into one consistent format
- Validates email addresses
- Removes duplicate rows (even when formatted differently)
- Splits results into:
  - `cleaned_output.csv` — ready-to-use clean data
  - `issues_report.csv` — rows flagged for human review (bad/missing data)

## Files
- `clean_data.py` — the automation script
- `messy_sample.csv` — sample messy input (8 rows, realistic issues)
- `cleaned_output.csv` / `issues_report.csv` — generated by running the script

## Run it yourself
```bash
python clean_data.py
```

## Real-world use cases for clients
- Cleaning CRM exports before a bulk import (HubSpot, Salesforce, Zoho)
- Merging multiple lead lists while removing duplicates
- Prepping raw scraped/collected data for outreach campaigns
- Standardizing large Excel sheets before analysis
